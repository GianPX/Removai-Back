## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Running AI API
This must be up
 ```bash
 $ cd ai-api
 #Run cli as admin if installation doesn't work
 $ pip install Flask
 #Run api
 $ python main.py
 ```
